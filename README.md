# Java Calculator App

A simple GUI-based calculator built with Java Swing as a class project to practice object-oriented programming and event-driven design.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Decimal point support
- Negative number toggle
- Delete and Clear buttons

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `Calculator.java`

## Built With
- Java
- Java Swing (GUI)
